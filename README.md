

# 🎸 MusicFIR (Flutter App)

Мобильное приложение интернет-магазина музыкальных инструментов, разработанное на Flutter с использованием Firebase и Provider.

---

## 📱 Описание проекта

Приложение позволяет пользователям:

- просматривать каталог музыкальных инструментов
- фильтровать товары по параметрам (категория, цена, рейтинг и т.д.)
- сравнивать выбранные товары
- настраивать инструмент в карточке товара (конструктор)
- переключать светлую и тёмную тему интерфейса
- получать персонализированный интерфейс с сохранением настроек

---

## ⚙️ Основной функционал

### 🛍 Каталог товаров

- список музыкальных инструментов
- динамическая загрузка данных
- отображение карточек товаров

### 🔎 Фильтрация

- фильтр по категории
- фильтр по цене
- фильтр по рейтингу
- комбинированная фильтрация

### ⚖️ Сравнение товаров

- добавление товаров в список сравнения
- просмотр характеристик нескольких инструментов
- сохранение списка сравнения

### 🎛 Конструктор инструмента

- настройка параметров инструмента в карточке товара
- выбор характеристик (цвет, аксессуары и др.)
- формирование итоговой конфигурации

### 🌗 Тёмная тема

- переключение светлой/тёмной темы
- сохранение выбора пользователя
- синхронизация состояния интерфейса

---

## 🧱 Архитектура

Приложение построено по архитектуре:

- **MVVM (Model–View–ViewModel)**
- **Provider** для управления состоянием
- **Repository pattern** для работы с Firebase

---

## 🔥 Используемые технологии

- Flutter
- Dart
- Firebase (Firestore)
- Provider
- SharedPreferences
- Material Design

---

## 🧠 Особенности реализации

- реактивное обновление UI через Provider
- разделение логики и интерфейса (MVVM)
- хранение пользовательских настроек (тема, сравнение)
- работа с облачной базой данных Firebase
